# 📊 Sales Performance Dashboard (Power BI)
## Dashboard Preview
![Dashboard](Dashboard.png)

## Overview
This project presents an interactive Power BI dashboard created using a sales dataset. The dashboard helps analyze sales performance, trends, and key business insights.

## Dataset
- `Sales Data.xlsx` – Source dataset used to build the dashboard

## File Included
- `Sales_Dashboard.pbix` – Power BI report file

## Key Features
- Sales trend analysis
- Region-wise performance insights
- Category and product-level breakdown
- Interactive filters and visuals

## Tools Used
- Microsoft Power BI
- Microsoft Excel

## How to Use
1. Download both files from this repository
2. Open the `.pbix` file in Power BI Desktop
3. If required, reconnect the dataset (`Sales Data.xlsx`)
4. Explore the dashboard using filters

## Project Objective
To convert raw sales data into meaningful insights using data visualization techniques.
